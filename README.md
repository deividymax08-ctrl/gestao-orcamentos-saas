# Gestão de Orçamentos SaaS

Sistema completo para criar, editar e gerenciar orçamentos.  
Painel admin + app cliente + salvamento local no celular/computador.

### O que faz
- Login obrigatório (senha gerada no painel)  
- Controle de acesso: ativo / bloqueado / pagamento pendente  
- Pagamento PIX gerado no painel  
- Atualização automática do app via GitHub  
- Dados do usuário salvos localmente (pasta GestaoOrcamentosApp)  
- Funciona offline depois do login
